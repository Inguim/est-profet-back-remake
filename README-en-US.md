# Profet: Backend Remake

> Original Project [Profet](https://github.com/Inguim/profet)

The purpose of this project is to rewrite an old project that was originally built using Laravel and MVC architecture. However, bringing to Node.Js and TypeScript with quality-of-life improvements, new implementation techniques for some features, and the addition of ideas that were not possible in the original project. In addction, this remake will contain two branches: **<code>mvc</code>** and **<code>ca</code>**.

- MVC: responsible for containing a version based on MVC architecture, staying faithful to the original projetct.
- CA: responsable for containing a version that follows Clean Architecture principles.

> Both versions are just a way to **pay respect** to the original project that I developed at the begining of my career, **with no intention of monetizing it or releasing it for public use**
