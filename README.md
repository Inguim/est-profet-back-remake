# Profet: Backend Remake

> Read in [English](https://github.com/Inguim/est-profet-back-remake/tree/main/README-en-US.md)
> Projeto original [Profet](https://github.com/Inguim/profet)

O intuito desse projeto e reescrever um antigo projeto que está com a sua codebase em Laravel com arquitetura MVC, porém trazendo para NodeJs + Ts com melhorias de qualidade de vida, novas técnicas na implementação de algumas features e adição de algumas ideias que não foram possíveis na época de sua produção. Além disso, o _remake_ contará com duas branchs: **<code>mvc</code>** e **<code>ca</code>**.

- MVC: branch responsável por conter um versão escrita na arquitetura de MVC respeitando o projeto original.
- CA: branch responsável por conter uma versão escrita com os principios de Clean Architecture.

> Ambas versões são apenas uma forma de **prestar tributo** ao primeiro projeto que desenvolvi em minha carreira, **sem intenções de ganhos ou de ter essa versão disponibilizada do produto para uso**.
